Что есть сейчас:

Сейчас сайт представляет из себя задачник, который позволяет пользовать нескольким пользователям добавлять задачи на главный экран, редактировать, удалять.

Каждый пользователь имеет свой личный кабинет, со своими данными авторизации

Планы

1. Задеплоить проект

2. Создать возможность добавления неограниченного количества строк в проект

3. Используя Mimesis - Fake Data Generator заполнять таблицы данными

4. Добавить возможность выгрузки таблиц в формате CSV

5. Разграничить права доступа пользователей

Технологии и интрументы, которые использовались при разработке

1. Django как основной инструмент проекта

2. SQLite как основную СУБД локально, PostgreSQL

3. Знания HTML и CSS для внешнего вида сайта

4. Шаблонизатор jinja2

5. Heroku для деплоя

6. Mimesis - Fake Data Generator для генерации данных в таблицы

6. GITHUB для контроля версий
